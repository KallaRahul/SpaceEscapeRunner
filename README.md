# Space Escape Runner 🚀

A cyberpunk-themed, infinite 2D space runner game built with React Native and Expo.

## How to Run the Game Locally

1. **Clone the repository** (if downloading from GitHub):
   ```bash
   git clone <your-github-repo-link>
   cd SpaceRunnerGame
   ```

2. **Install the dependencies**:
   Make sure you have Node.js installed, then run:
   ```bash
   npm install
   ```

3. **Start the Expo server**:
   ```bash
   npx expo start
   ```

4. **Play on Mobile (Recommended)**:
   - Download the **Expo Go** app from the iOS App Store or Google Play Store.
   - Scan the QR code that appears in your terminal.
   - Enjoy the neon aesthetic and smooth 60fps gameplay!

5. **Play on Web**:
   - In the terminal menu, press `w` to open the game in your local web browser.
